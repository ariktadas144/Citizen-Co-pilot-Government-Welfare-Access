# Citizen-Co-pilot-Government-Welfare-Access
# Welfare Access Website

## Problem Statement
Many citizens are unaware of government welfare schemes or find it difficult to access them.

## Solution
A single platform that helps users discover and understand welfare schemes based on eligibility.

## Features
- Scheme discovery
- Eligibility filtering
- Simple UI
- Mobile-friendly design

## Tech Stack
- HTML, CSS, JavaScript
- Python (Flask) / Node.js
- GitHub

## How to Run
1. Clone the repository
2. Open frontend/index.html
3. Run backend server
